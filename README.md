## 🎯 About The Project

Ini adalah situs web portofolio pribadi yang dirancang dan dikembangkan oleh Daffa Naufal Fachrezi. Situs ini menampilkan karya profesional, keterampilan, dan proyek-proyek dalam desain yang bersih, modern, dan responsif.

### Objectives:
- Showcase professional portfolio
- Provide contact form integration with EmailJS
- Display projects and certifications
- Smooth animations and transitions
- Fully responsive across all screen sizes

---

## ✨ Features

- 📱 **Fully Responsive Design
- 🎨 Modern, Stylish & Minimalist Aesthetic
- 📧 EmailJS Integration for Contact Form
- 📜 Certificate and Project Gallery with Lightbox
- 📊 Coming Soon Page with Animated Intro
- 🔍 Scroll Animations with AOS Library
- 📱 Mobile-first Approach
- ⚡ Optimized Performance

## 🛠️ Built With

* [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) - Structure
* [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) - Styling
* [JavaScript (Vanilla)](https://developer.mozilla.org/en-US/docs/Web/JavaScript) - Interactivity
* [Ion Icons](https://ionic.io/ionicons) - Icons
* [EmailJS](https://www.emailjs.com/) - Contact Form Integration
* [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) - Scroll Animations
* [Google Fonts](https://fonts.google.com/) - Typography
